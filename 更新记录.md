### Props
|  属性名  | 类型 | 默认值 | 说明 |
|  :----:  | :----:  | :----: | :----: |
| min  | Number | 最小值 | 0 |
| max  | Number | 最大值 | 0 |
| defaultValue  | Number | 默认值 | 0 |
